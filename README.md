Forge is an enterprise project planning and workflow automation platform.

Its main purpose is to automate the creation and maintenance of large-scale project plans used across healthcare and business initiatives.

The platform integrates with systems like:

Aha,
Rally/Icarus,
Smartsheet,
and MongoDB

to collect project information, process workflows, and automatically generate project plans.

Instead of teams manually creating and updating large Smartsheet project plans, Forge automates much of that process by:

pulling project data,
applying business rules,
generating structured plans,
and synchronizing updates across systems.

Forge also acts as a centralized operational platform where project metadata, workflows, integrations, and planning logic are managed together.

Recently, an AI-powered knowledge assistant capability was added into Forge to help users interact with the platform more naturally by asking questions about:

workflows,
project plans,
integrations,
deployment processes,
documentation,
and enterprise data.

So overall, Forge is essentially a centralized enterprise project automation and workflow management platform enhanced with AI capabilities.
